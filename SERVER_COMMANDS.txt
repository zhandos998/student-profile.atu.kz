Check Docker on server
----------------------
  docker --version
  docker compose version

If Docker is not installed on Ubuntu:

  sudo apt update
  sudo apt install -y ca-certificates curl gnupg
  sudo install -m 0755 -d /etc/apt/keyrings
  curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
  sudo chmod a+r /etc/apt/keyrings/docker.gpg
  echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo $VERSION_CODENAME) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  sudo apt update
  sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
  sudo usermod -aG docker zhagparovr

After usermod, reconnect to SSH.


First deploy from Git
---------------------
Prepare project folder on server:

  ssh zhagparovr@172.20.252.171
  sudo mkdir -p /opt/apps/student_profile
  sudo chown -R zhagparovr:zhagparovr /opt/apps/student_profile
  rm -rf /opt/apps/student_profile/*
  git clone --branch main https://github.com/zhandos998/student-profile.atu.kz.git /opt/apps/student_profile


Create .env.docker on server
----------------------------
  cd /opt/apps/student_profile
  cp .env.docker.example .env.docker
  nano .env.docker

Set real values:
  APP_URL=https://student-profile.atu.kz
  DB_DATABASE=student_profile
  DB_USERNAME=student_profile
  DB_PASSWORD=<strong_mysql_password>
  MYSQL_ROOT_PASSWORD=<strong_mysql_root_password>

Generate APP_KEY:
  docker compose --env-file .env.docker -f docker-compose.prod.yml run --rm --no-deps --build app php artisan key:generate --show

Paste the generated key into .env.docker:
  APP_KEY=base64:...


Build and start Docker
----------------------
  cd /opt/apps/student_profile
  docker compose --env-file .env.docker -f docker-compose.prod.yml up -d --build


Run Laravel setup
-----------------
  cd /opt/apps/student_profile
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec app php artisan migrate --force
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec app php artisan db:seed --force
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec app php artisan optimize:clear
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec app php artisan config:cache
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec app php artisan view:cache


Nginx reverse proxy on server
-----------------------------
Create config:

  sudo nano /etc/nginx/sites-available/student-profile.atu.kz

Paste:

  server {
      listen 80;
      server_name student-profile.atu.kz;

      client_max_body_size 128M;

      location / {
          proxy_pass http://127.0.0.1:8111;
          proxy_set_header Host $host;
          proxy_set_header X-Real-IP $remote_addr;
          proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
          proxy_set_header X-Forwarded-Proto $scheme;
      }
  }

Enable:

  sudo ln -s /etc/nginx/sites-available/student-profile.atu.kz /etc/nginx/sites-enabled/student-profile.atu.kz
  sudo nginx -t
  sudo systemctl reload nginx

Database
--------
Open MySQL CLI:
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec mysql mysql -u student_profile -p student_profile

Backup database:
  docker compose --env-file .env.docker -f docker-compose.prod.yml exec mysql mysqldump -u student_profile -p student_profile > student_profile-backup.sql
